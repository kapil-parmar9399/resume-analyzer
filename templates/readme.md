# AI Resume Analyzer

AI Resume Analyzer is a web application that analyzes resumes and provides feedback based on job descriptions. It helps users improve their resumes by checking keywords, skills, and ATS compatibility.

## 🚀 Features

* Upload Resume (PDF)
* Resume Score Analysis
* Keyword Matching with Job Description
* Skills Detection
* Resume Improvement Suggestions
* Download Report

## 🛠 Tech Stack

* Python
* FastAPI
* HTML / CSS / Bootstrap
* NLP (spaCy / ML models)
* MongoDB

## ⚙ Installation

Clone the repository:

```
git clone https://github.com/kapil-parmar9399/resume-analyzer.git
```

Install dependencies:

```
pip install -r requirements.txt
```

Run the application:

```
uvicorn app:app --reload
```

Open browser:

```
http://127.0.0.1:8000
```
